# is Valid ZIP Code

Validation for ZIP Code input.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
var zipCodeInput = getElementById(#contact-zipcode);
isValid.zipcode(zipcodeInput);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
